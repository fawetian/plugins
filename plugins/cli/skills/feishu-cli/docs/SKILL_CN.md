# feishu-cli — 飞书平台命令行工具

`feishu-cli` 已安装在本地并可在 PATH 中使用。当用户需要与飞书（Lark）开放平台交互时，使用此工具。

## 自主探索协议

**执行命令前，始终通过 `--help` 发现用法，不猜测参数名称。**

```
# 第一步：查看顶层命令
feishu-cli --help

# 第二步：查看某模块的子命令
feishu-cli <command> --help

# 第三步：查看具体操作的参数
feishu-cli <command> <subcommand> --help
```

先运行这些命令读取输出，再构造实际命令。

## 配置方式

CLI 需要飞书应用凭证，按以下顺序检查：

1. **环境变量**（推荐）：
   ```
   FEISHU_APP_ID=cli_xxx
   FEISHU_APP_SECRET=xxx
   ```
2. **配置文件**：`~/.feishu-cli/config.yaml`
3. 若两者都不存在，提示用户先配置凭证再操作。

调试时使用 `--debug` 参数查看 HTTP 请求/响应详情。

## 模块索引

以此为起点选择正确模块，再通过 `--help` 了解具体参数：

| 模块 | 用途 |
|------|------|
| `doc` | 创建、获取、编辑、导出/导入文档，添加高亮块 |
| `wiki` | 知识库——获取节点、列出空间、导出文档 |
| `file` | 云空间——列出、创建、移动、复制、删除文件 |
| `msg` | 发送消息、搜索群聊、查看会话历史 |
| `task` | 创建、查看、更新、完成任务 |
| `tasklist` | 任务清单管理 |
| `calendar` | 日历和日程管理 |
| `comment` | 列出、添加、删除文档评论 |
| `perm` | 添加或更新文档权限 |
| `user` | 获取用户信息 |
| `board` | 画板操作 |
| `media` | 上传/下载素材 |
| `sheet` | 电子表格操作 |
| `search` | 搜索消息和应用（需要用户授权） |
| `auth` | 用户授权管理 |
| `dept` | 部门操作 |
| `chat` | 群聊管理 |
| `config` | 初始化和管理 CLI 配置 |

## 使用原则

1. **始终先运行 `--help`** — 不猜测参数或子命令名称。
2. **破坏性操作前确认** — 删除文件、移除权限等操作前需用户确认。
3. **展示命令输出** — 每次命令执行后，清晰展示结果给用户。
4. **优雅处理错误** — 命令失败时，展示错误信息并建议下一步（检查凭证、检查 ID、使用 `--debug`）。
5. **文档 ID** — 许多操作需要文档/节点/用户 ID；如未提供，询问用户。
