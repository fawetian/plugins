# Code Simplifier

在保持功能完全不变的前提下，自动简化和优化刚编写或修改的代码，提升代码的清晰度、一致性和可维护性。

## 安装

```bash
/plugin install coding@fawetian-plugins
```

## 触发方式

该 skill 以**自主模式**运行，在代码编写或修改后自动执行，无需显式触发。

你也可以在对话中明确请求：

- `帮我简化这段代码`
- `优化这个函数的可读性`
- `重构最近修改的代码`

## 使用示例

**场景一：自动简化刚写完的代码**

编写完一段功能代码后，skill 会自动分析是否存在不必要的嵌套、冗余逻辑或命名不清晰的问题，并提出改进建议，保持功能完全等价。

**场景二：消除嵌套三元运算符**

当代码中出现嵌套三元运算符时，skill 会将其重构为更易读的 `if/else` 链或 `switch` 语句，避免阅读困难。

**场景三：遵循项目规范**

自动读取项目 `CLAUDE.md` 中的编码规范，确保简化后的代码符合项目约定的风格和标准，而不是通用的最佳实践。

## 输出

直接对当前编辑的代码文件进行修改，不生成额外文件。每次重构仅处理本次会话中最近修改的代码范围，除非明确指示扩大范围。修改内容会有简要说明，解释关键改动对可读性的影响。
