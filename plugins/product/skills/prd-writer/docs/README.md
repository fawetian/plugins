# PRD 写作助手

专业的产品需求文档（PRD）写作助手，帮助你快速生成结构规范、内容完整的 PRD 文档。

## 安装

```bash
/plugin install product@fawetian-plugins
```

## 触发方式

在对话中使用以下任意触发词即可激活该 skill：

- `prd`
- `product requirement`
- `write prd`
- `create prd`
- `feature spec`

例如：输入 "帮我写一份用户登录功能的 prd" 即可触发。

## 使用示例

**示例 1：新功能 PRD**

```
帮我写一份用户登录功能的 PRD，包括手机号登录和第三方登录
```

**示例 2：从需求背景开始**

```
我们要做一个商品推荐系统，请帮我 write prd，背景是用户在浏览商品时缺乏个性化推荐
```

**示例 3：整理零散需求**

```
请基于以下需求描述 create prd：我们需要一个消息通知中心，支持系统通知、活动通知和个人消息分类管理
```

## 输出

生成的 PRD 文档保存至 `docs/prd/{feature-name}.md`，包含以下标准章节：

- 文档头部（标题、版本、作者、状态）
- 背景与问题陈述
- 目标与成功指标（OKR）
- 用户故事与使用场景
- 功能需求（含 P0/P1/P2 优先级）
- 非功能需求
- 技术考量
- 时间线与里程碑
- 待确认事项

所有内容均以中文输出，流程图使用 Mermaid 语法绘制。
